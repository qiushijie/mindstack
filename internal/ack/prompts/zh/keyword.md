# Keyword Extraction

从用户问题中提取有实质含义的词组作为关键词，同时提供英文等价词，用于全文搜索。

---

你正在从用户问题中提取关键词，用于在知识库文档中进行全文搜索。知识库文档混合了中文和英文内容，因此需要同时提供两种语言的关键词。

用户问题: %s

请从问题中提取 3-8 个有实质含义的中文词组作为关键词，并额外提供 2-5 个对应的英文关键词:
- 中文关键词必须是完整的词组，不能拆成单字（例如"权限控制"应提取为"权限"和"控制"，而不是"权""限""控""制"）
- 去除虚词（如"的""了""如何""什么""可以"等）
- 英文关键词为中文关键词对应的英文技术术语、同义词或相关词
- 英文关键词用于匹配英文标签、别名、摘要和全文内容

只返回 JSON 字符串数组（不要 markdown 围栏）。示例:["权限","控制","permission","access-control","rbac"]
