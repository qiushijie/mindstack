# 关键词提取

从用户问题中提取有实质含义的复合词组和领域专有名词作为关键词，同时提供英文等价词，用于全文搜索。

---

你正在从用户问题中提取关键词，用于在知识库文档中进行全文搜索。知识库文档混合了中文和英文内容。

用户问题: %s

请从问题中提取 3-8 个关键词，要求:
- 优先提取完整的复合术语和领域专有名词（例如"权限控制"应提取为"权限控制"，"指数退避"应提取为"指数退避"）
- 如果问题中包含英文技术术语，直接保留原文（例如 "RBAC", "Kubernetes HPA"）
- 提取对应英文技术术语的原文形式（例如 "retry policy" 对应 "重试策略"）
- 去除无区分度的虚词和通用词（如 "的""了""如何""什么""可以""system""approach""method"）
- 不要将复合词拆成单字

只返回 JSON 字符串数组（不要 markdown 围栏）。示例:["权限控制","access-control","rbac","retry-policy","指数退避"]
