# Rerank

根据用户问题从候选文档中选出最相关的文档。

---

你正在帮助从候选文档中筛选出与用户问题最相关的文档。每个文档包含元信息和正文预览。注意：某些文档的正文可能被截断（标注了总行数但只显示了前50行），关键内容可能在预览范围之外。

用户问题: %s

候选文档:
%s

请从中选出最多 %d 篇最相关的文档，按相关度降序排列。
对每篇文档给出 0-1 之间的相关度分数。

评分建议:
- 0.8-1.0：文档直接回答了用户问题
- 0.5-0.7：文档包含相关信息但需要进一步提取
- 0.3-0.4：文档主题相关但可能不直接回答问题
- 0.0-0.2：几乎不相关

只返回 JSON 数组（不要 markdown 围栏）:
[
  {"path": "doc.md", "score": 0.95}
]

如果没有相关文档，返回 []。
