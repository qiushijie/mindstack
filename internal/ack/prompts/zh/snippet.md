# Snippet Extraction

从单篇 markdown 文档中提取最能回答用户问题的连续行区间。

---

你正在从一篇 markdown 文档中提取证据,用来帮助回答用户的问题。

用户问题:%s
文档路径:%s

文档内容(每行前面已加上行号前缀):
%s

请挑选最多 %d 段连续的行区间,要最能回答该问题。每段给出 0-1 之间的相关性得分。

只返回一个 JSON 对象(不要带 markdown 围栏):
{
  "snippets": [
    {"start": 10, "end": 25, "score": 0.9}
  ]
}

如果文档与问题无关,返回 {"snippets":[]}。
