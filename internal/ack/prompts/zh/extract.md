# Snippet Extraction

从文档中提取与用户问题最相关的片段。

---

你正在从一篇文档中提取与用户问题最相关的片段。

用户问题: %s

文档内容（行号:内容）:
%s

请提取最多 %d 个与用户问题最相关的连续片段。要求:
- 片段必须是文档中连续的行
- 每个片段长度不超过 15 行
- 优先选择直接回答问题的内容，其次是提供背景或相关细节的内容
- 对每个片段给出 0-1 之间的相关度分数

只返回 JSON 数组（不要 markdown 围栏）:
[
  {"startLine": 10, "endLine": 20, "score": 0.95}
]

如果没有相关内容，返回 []。
